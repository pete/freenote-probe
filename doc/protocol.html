HEADER FIELDS msg type 2 byte code for message type
0x0001 = first checkin
0x0002 = give work
0x0003 = report work
0x0101 = assign probe id
0x0102 = work packet
0x0103 = save result success
0x01FF = error
probe id 4 byte integer unique identifier of probe
probe ver 2 byte integer version of the probe code
proto ver 2 byte integer version of the protocol version
size 2 byte integer of size in bytes of the message body
sig 48 byte signature of the contents of the message body
Message 1 Probe checks in for first time
header msg type = 0x0001
body team_name=[a-z\d]+ pass over our team name
pin_code=\d{4}
pub_key=$KEY pass over our PEM public key
os_type=$OS text OS string
Result Success
header msg type = 0x0101
body probe_id=\d+ assign him a probe id
dispatch_key=$KEY give him our PEM public key
Result Failure
header msg type = 0x01FF
body error=$ERROR_MESSAGE
Message 2 Probe asks for work
header msg type = 0x0002
body time=\d+ we need some filler
Result Success
header msg type = 0x0102
body work_id=\d+ the unique ID of this work
$URLID=$TYPE,$ACTION,$URL can repeat this line N times
capabilities=X capabilities of the probe*
URL WORK:
$URLID unique integer ID of this url
$TYPE 1 => http, 2 => https
$ACTION 1 => GET
$URL  full url to probe
capabilities:
a bit flag of the $TYPEs he can support (i.e 1 || 2 = 3 means http and httpd)
Result Failure
header msg type = 0x01FF
body error=$ERROR_MESSAGE
Message 3 Report Results to collection server
header msg type = 0x0003
body work_id=\d+ work id we completed
$URLID=$TYPE,$RESULT_CODE,$HTTP_RESULT,$IP_RESOVED,$SIZE_BYTES,$DNS_MS,
$REDIRECT_MS,$CONNECT_MS,$FIRST_BYTE_MS,$PAGE_MS,
$COMPONENT_MS,$NUM_COMPONENTS,$TOTAL_MS
metrics=$UPTIME, $LOAD1, $LOAD5, $LOAD15, $CPU_MS
RESULTS:
$URLID unique url ID from the work packet
$TYPE type code from the work packet
$RESULT_CODE 1 => success, 2 => dns error, 3 => http timeout, 4 => other error
$HTTP_RESULT http result code (200, 302, etc)
$IP_RESOVED text ip address (ie 127.0.0.1)
$SIZE_BYTES integer size of bytes downloaded
$DNS_MS integer num of ms to retrieve dns address
$REDIRECT_MS integer numof ms to be redirected
$CONNECT_MS Integer num of ms until connection
$FIRST_BYTE_MS Integer num of ms to receive first byte
$PAGE_MS Integer num of ms to receive page
$COMPONENT_MS Integer num of ms to download all compoents
$TOTAL_COMPONENTS Integer number of components downloaded
$COMPONENT_SIZE_BYTES integer size of component bytes downloaded
$TOTAL_MS Intger ms to complete request
metrics:
$UPTIME seconds of uptime
$LOAD1 1 minute load average
$LOAD5 5 minute load average
$LOAD15 15 minute load average
$CPU_MS proprietary cpu timing in ms
